Această aplicație este un serviciu web 
de învățare automată care prezice 
rezultate (cum ar fi supraviețuirea în 
setul de date Titanic) folosind un model 
antrenat cu Ludwig, un framework de 
învățare automată. Nucleul 
aplicației este un server Flask care 
permite utilizatorilor să trimită date 
de intrare printr-un punct final HTTP 
și să primească predicții ca răspuns. 
Pentru a monitoriza comportamentul 
aplicației, Prometheus este integrat 
pentru a urmări indicatori precum 
numărul de predicții făcute.


Am învățat cum să integrez modele de 
învățare automată în aplicații web și 
să le monitorizez folosind instrumente 
precum Prometheus. De asemenea, am 
acumulat experiență lucrând cu Flask și 
Ludwig.


Cea mai dificilă parte a fost probabil 
integrarea Grafana.
